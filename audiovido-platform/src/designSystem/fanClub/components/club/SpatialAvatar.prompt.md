# SpatialAvatar

A member placed at an absolute room coordinate (bar stool, cinema seat). The glow ring intensity tracks their `activity`; when they post, you can pass a `speech` string and a bubble pops above them.

```jsx
<SpatialAvatar name="Reni Q" top="64%" left="22%" activity={0.85}
               speech="that drum break is unreal" />
```
