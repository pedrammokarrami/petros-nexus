# Friends Page — Design System

Social networking UI kit for the Nexus platform. Green-accented, glassmorphic, responsive.

## Tokens
- colors.css — void-dark palette + green accent (#4ADE80)
- typography.css — Inter + JetBrains Mono
- spacing.css — 4-based scale
- motion.css — durations, easing, keyframes

## Components
- GlowAvatar — circular avatar with online glow
- Badge — small labeled chip
- Button — primary/secondary/ghost/icon variants
- Card — glass card wrapper
- TabBar — tab navigation with glass styling
- Pill — stat pill for compatibility
- Input — glass search input

## UI Kits (friends-page)
- Header — logo + title + tabs
- FriendsTab — filter + avatar grid
- DiscoverTab — swipe card + suggestion rail
- SearchTab — search input + results
- ReferralsTab — referral list + share code
- FriendProfileModal — full profile overlay
- DiscoverDeck — swipeable card stack
- SearchScreen — search + result grid
- SuggestionRail — horizontal scroll carousel
