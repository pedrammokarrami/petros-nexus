# PhysicalButton

Action buttons styled as physical venue objects. Four variants:

```jsx
<PhysicalButton variant="drink"   label="Order"     onClick={openMenu} />
<PhysicalButton variant="jukebox" label="Vote"      icon="VOTE" />
<PhysicalButton variant="arcade"  label="Challenge" icon="⚡" />
<PhysicalButton variant="neon"    label="React"     icon="😍" />
```

Each picks up the room's `--accent` automatically.
