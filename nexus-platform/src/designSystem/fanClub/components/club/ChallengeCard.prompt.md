# ChallengeCard

Side-panel widget showing the room's active challenge. Glowing progress bar pulls the accent from the current club scope.

```jsx
<ChallengeCard
  title="Drop a song for the rain"
  description="Share one track that fits a gray afternoon."
  progress={0.34}
  progressLabel="17 of 50 shared"
  tag="Daily"
  onJoin={() => {}}
/>
```
